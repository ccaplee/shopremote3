# ShopRemote2 기여 가이드

ShopRemote2에 기여해 주셔서 감사합니다!

## 기여 방법

1. 이 저장소를 포크합니다
2. 기능 브랜치를 생성합니다 (`git checkout -b feature/새기능`)
3. 변경사항을 커밋합니다 (`git commit -m '새기능 추가'`)
4. 브랜치에 푸시합니다 (`git push origin feature/새기능`)
5. Pull Request를 생성합니다

## 개발 환경 설정

- Rust 개발 환경
- Flutter SDK 3.22.3
- C++ 빌드 도구

자세한 빌드 방법은 [README](../README.md)를 참고하세요.
