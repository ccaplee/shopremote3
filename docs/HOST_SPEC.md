## 제품 개요

### 비전
**ShopRemote2 Host**는 점포(매장), 편의점, 카페 등 소규모 매장의 매출 관리, POS 시스템, 보안 카메라 등 중요한 기기를 **원격으로 지원받을 수 있도록 하는 호스트 전용 솔루션**입니다.

TeamViewer Host, Chrome Remote Desktop Host와 유사하게:
- **호스트 역할만 수행** (원격 제어만 받음, 다른 장치 제어 불가)
- **무인 접근(Unattended Access)** 지원 (서비스 형태로 자동 실행)
- **극도로 단순화된 인터페이스** (설정 후 백그라운드에서 실행)
- **매장 담당자가 쉽게 설치 및 관리**할 수 있도록 설계

### 사용 사례
- **POS 시스템 원격 지원**: 매장의 POS 기계 장애 시 원격으로 진단 및 수정
- **보안 감시**: 점포 관리자가 다른 지점의 상황을 원격으로 모니터링
- **IT 헬프데스크**: 다수의 매장 기기를 중앙에서 원격 지원
- **마케팅 캠페인**: 매장 디지털 사이니지 원격 업데이트
- **온보딩 지원**: 신입 직원 또는 신규 매장 초기 설정 지원

## 타겟 사용자 및 시장

### 1차 타겟
- **소규모 매장/점포 운영자**: 기술에 익숙하지 않은 사용자
- **IT 지원팀**: 다수의 원격 사이트를 효율적으로 관리
- **프랜차이즈 본사**: 가맹점 기기의 원격 지원 인프라 필요

### 2차 타겟
- **대형 소매점 POS 지원팀**
- **의료시설 의료기기 원격 지원**
- **교육기관 컴퓨터실 관리**

## 주요 특징

### 1. 무인 접근 (Unattended Access)
- **시스템 서비스로 자동 실행** (Windows Service, systemd, launchd)
- 사용자 로그인 없이도 원격 접근 가능
- 호스트 기기의 화면 잠금 상태에서도 제어 가능
- 부팅 시 자동 시작

### 2. 간편한 식별
- **고유한 Device ID** (12자리 숫자)
- **자동 생성 보안 비밀번호** (6-10자리 숫자)
- QR 코드로 ID 및 패스워드 쉽게 공유
- 비밀번호 자동 변경 옵션 (시간 기반)

### 3. 보안
- **AES-256 기반 암호화** (네트워크 통신)
- **2FA 지원** (선택사항)
- **IP 화이트리스트** 설정
- **연결 로그** 기록 및 감시
- **권한 제어** (화면 보기만, 파일 전송, 클립보드 등)

### 4. 극도로 단순화된 UI
- **시스템 트레이 아이콘** (Windows/Linux)
- **메뉴바 아이콘** (macOS)
- **최소한의 설정창** (Device ID, Password, Status만 표시)
- **타이머 기반 자동 시작** 옵션

### 5. 확장 가능성
- 향후 모바일 앱 지원 가능 (iOS/Android)
- 커스텀 브랜딩 옵션
- 플러그인 시스템 (향후)

## 기능 명세

### 호스트 기능 (KEEP)

#### A. 원격 화면 공유
- [ ] **화면 캡처** - 호스트 화면 실시간 스트리밍
  - 다중 모니터 지원
  - 화면 해상도 자동 감지
  - 프레임 레이트 최적화 (기본 30fps, 설정으로 60fps)
  - 저역폭 모드 (해상도 자동 조정)

- [ ] **원격 마우스/키보드 제어**
  - 마우스 포인터 이동 및 클릭
  - 키보드 입력
  - 스크롤 휠 지원
  - 멀티터치 제스처 (향후)

- [ ] **클립보드 공유**
  - 텍스트 복사/붙여넣기
  - 이미지 복사 (선택사항)
  - 파일 경로 공유 (선택사항)

#### B. 파일 전송
- [ ] **양방향 파일 전송**
  - 컨트롤러 → 호스트
  - 호스트 → 컨트롤러
  - 드래그 앤 드롭 지원
  - 폴더 재귀 전송
  - 대용량 파일 이어받기(Resume) 지원

- [ ] **파일 브라우저**
  - 호스트 파일 시스템 탐색
  - 파일 삭제, 이름 변경 (권한 기반)
  - 최근 파일 목록

#### C. 기본 시스템 정보
- [ ] **디바이스 정보 표시**
  - Device ID (고유 식별번호)
  - Hostname
  - OS 정보 (Windows/macOS/Linux)
  - IP 주소 (로컬 및 공인 IP)
  - MAC 주소

- [ ] **상태 표시**
  - 온라인/오프라인 상태
  - 현재 연결 상태
  - 마지막 연결 시간
  - 활성 세션 수

#### D. 연결 관리
- [ ] **동시 연결 제한**
  - 기본 1개 세션 (설정으로 변경 가능)
  - 다중 세션 거부 옵션

- [ ] **연결 권한 제어**
  - 특정 사용자만 접근 허용
  - IP 주소 기반 필터링
  - 접근 토큰(Token) 기반 인증

- [ ] **연결 로깅**
  - 모든 연결 시도 기록
  - 연결 시작/종료 시간
  - 연결한 사용자 정보
  - 수행한 작업 기록 (선택사항)

#### E. 보안 설정
- [ ] **비밀번호 관리**
  - 사용자 정의 비밀번호 설정
  - 자동 생성 비밀번호
  - 비밀번호 주기적 변경 옵션
  - 비밀번호 복잡도 설정

- [ ] **권한 세분화**
  - 화면 보기만
  - 화면 보기 + 입력 제어
  - 파일 전송 권한
  - 클립보드 권한

- [ ] **암호화**
  - AES-256 기반 전송 암호화
  - TLS/SSL 지원 (향후)

#### F. 시스템 통합
- [ ] **서비스 시작/중지**
  - 시작/중지 버튼
  - 서비스 상태 표시
  - 오류 로그 표시

- [ ] **자동 시작 설정**
  - 부팅 시 자동 실행
  - 서비스 자동 시작
  - 지연 시작 옵션 (설정 가능)

- [ ] **시스템 트레이 통합**
  - 최소화 시 트레이로
  - 트레이 아이콘 클릭으로 창 열기
  - 빠른 접근 메뉴

#### G. 네트워크 설정
- [ ] **서버 설정**
  - 기본 서버 (ShopRemote2 공식 서버)
  - 커스텀 서버 설정 가능
  - 서버 연결 상태 표시
  - 자동 재연결 (Reconnect)

- [ ] **포트 설정**
  - TCP 포트 커스터마이징 (기본 5900)
  - UDP 포트 설정
  - 포트 충돌 감지 및 자동 조정

- [ ] **방화벽 통과**
  - UPnP/NAT-PMP 자동 포트 포워딩
  - 프록시 설정 지원
  - 릴레이 서버 자동 선택

#### H. 성능 및 최적화
- [ ] **대역폭 조절**
  - 최대 대역폭 설정
  - 자동 품질 조정
  - 네트워크 상태에 따른 적응형 인코딩

- [ ] **CPU/메모리 최적화**
  - 저전력 모드 (배터리 절약)
  - 리소스 사용량 모니터링

#### I. 다국어 지원
- [ ] **한국어** (1차)
- [ ] **영어** (1차)
- [ ] **중국어** (2차)
- [ ] **일본어** (2차)

## 사용자 흐름

### Flow 1: 설치 및 초기 설정

```
시작
  ↓
설치 마법사 실행
  ├─ Device ID 할당 (자동)
  ├─ 비밀번호 생성 (자동)
  └─ QR 코드 표시
  ↓
초기 설정 (선택사항)
  ├─ 부팅 시 자동 시작 설정
  ├─ 서버 선택 (기본값 사용 권장)
  └─ 권한 설정 (기본값: 모두 활성화)
  ↓
서비스 시작
  ├─ 시스템에 등록
  ├─ 자동 시작 활성화
  └─ 시스템 트레이 아이콘 표시
  ↓
완료 - ID/Password 메모
```

### Flow 2: 원격 접근 (컨트롤러 관점)

```
모바일/데스크톱 앱에서 ID 입력
  ↓
ID로 호스트 검색
  ↓
비밀번호 입력 요청
  ↓
비밀번호 인증
  ↓
호스트 허가 확인 (선택사항)
  ↓
원격 화면 표시
  ├─ 마우스/키보드 제어
  ├─ 파일 전송
  └─ 클립보드 공유
  ↓
세션 종료
```

### Flow 3: 비밀번호 변경

```
설정 패널 열기
  ↓
"비밀번호 변경" 버튼 클릭
  ↓
현재 비밀번호 인증 (로컬)
  ↓
새 비밀번호 입력 (또는 자동 생성)
  ↓
변경 확인
  ↓
QR 코드 업데이트
```

### Flow 4: 서비스 재시작

```
설정 패널
  ↓
[다시 시작] 버튼 클릭
  ↓
서비스 중지 (기존 연결 종료)
  ↓
대기 (2초)
  ↓
서비스 시작
  ↓
상태 업데이트: 온라인 ✓
```

## 배포 및 설치

### 1. 설치 형식

#### Windows
- **MSI Installer** (.msi) - 권장
  - 자동 업데이트 지원
  - 시스템 서비스 등록
  - 제어판 통합
- **Portable EXE** (선택사항) - USB 실행용

#### macOS
- **DMG Image** (.dmg) - 권장
  - 드래그 앤 드롭 설치
  - Gatekeeper 호환
- **Homebrew** (향후) `brew install shopremote2-host`

#### Linux
- **DEB Package** (.deb) - Ubuntu/Debian
- **RPM Package** (.rpm) - CentOS/RHEL/Fedora
- **AppImage** (독립형) - 모든 배포판
- **AUR** (Arch Linux) 지원

### 2. 설치 과정

```
1. 다운로드
   └─ https://shopremote2.com/download/host

2. 실행
   └─ Windows: shopremote2-host-v1.0.msi
   └─ macOS: shopremote2-host-v1.0.dmg
   └─ Linux: shopremote2-host_1.0_amd64.deb

3. 설치 마법사
   ├─ 약관 동의
   ├─ 설치 경로 선택
   ├─ 시작 메뉴 생성
   └─ 마침 → 자동 실행

4. 초기 설정 (선택사항)
   ├─ Device ID 확인
   ├─ 비밀번호 보기/변경
   ├─ 부팅 시 자동 시작 설정
   └─ 닫기

5. 완료
   └─ 시스템 트레이에 아이콘 표시
   └─ 상태: 온라인 ✓
```

### 3. 자동 업데이트

```
매주 1회 (수요일 23:00)
├─ 서버에서 최신 버전 확인
├─ 새 버전 발견 시 알림
├─ 사용자 수락 후 다운로드
├─ 백그라운드에서 설치
├─ 다음 실행 시 적용
└─ 변경사항 알림 표시
```

### 부록 A: 용어 정의

| 용어 | 정의 |
|------|------|
| **Host** | 원격 제어를 받는 머신 (POS, 가게 PC 등) |
| **Controller** | 원격으로 Host를 제어하는 사용자 장치 |
| **Device ID** | Host의 고유 식별번호 (12자리 숫자) |
| **Relay Server** | 호스트와 컨트롤러 간 중개 서버 |
| **Incoming Only** | 호스트 역할만 (제어받기만, 제어하기 불가) |
| **Unattended Access** | 무인 접근 (사용자 상호작용 없이 원격 접근) |
| **UPnP** | Universal Plug and Play - 자동 포트 포워딩 |
| **2FA** | 2-Factor Authentication - 2단계 인증 |
